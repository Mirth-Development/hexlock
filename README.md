# HEXLOCK
Made for Wheel Jam 2, 2026.

Lockpicking game made for Wheel Jam 2, 2026. A project lovingly rushed by me (Gabe Allison) and Kira Stephenson. There is no win or lose screen, as we didnt have time lol. 


Lockpicking "Roguelike"​ game, or at least that was the aim. There is no wheel in this game because we were told that we could do a lockpick game. Yeah, we're rebels. Maybe the real "wheel" was the friends we made along the way.


Controls

    ​W - Key 
        ​​Electric pick (yellow) = Hold to charge
        Magic pick (purple) = Use arrow keys to do mini arrow game
        Regular(white) = Smack that mf thing in
        ​

    A -Key and D - Key = Move pick left and right

    Arrow keys (or J-Key, I-Key, L-key, K-Key) = Play small arrow game to activate the magic pick.

Tips

    Electric pick can stun set tumblers
    Rust can be banged off (and so can I)
    Tumblers must be set in order
